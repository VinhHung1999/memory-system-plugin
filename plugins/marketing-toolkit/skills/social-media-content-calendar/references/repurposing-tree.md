# Repurposing Tree — Core → Derivatives

Mapping from core content pieces to platform-native derivatives. Batch-produce cores, derive the rest.

## Blog post (1500-word article)

**Core:** Blog on your website.

**Derivatives (1 blog → 6+ social posts across 2-3 weeks):**
1. **Twitter thread** — 5-8 tweets summarizing key points; link to blog in final tweet
2. **LinkedIn post** — rewrite first 3 paragraphs as standalone native post; comment with blog link
3. **Instagram carousel** — 7-10 slides, one key point per slide, branded template
4. **Instagram Reel (15s)** — video of you narrating the top insight
5. **Email newsletter feature** — intro paragraph + link
6. **Facebook post** — shortened 100-word hook + link

## Long YouTube video (10-20 min)

**Core:** Long-form video.

**Derivatives (1 video → 10+ posts):**
1. **3-5 YouTube Shorts** — best 30-60s moments
2. **Instagram Reels (same shorts adapted)**
3. **TikTok versions** (adjust for vertical + trending audio)
4. **Twitter thread** — timestamps + key quotes
5. **LinkedIn post** — lessons learned from making it
6. **Instagram carousel** — 5 biggest takeaways
7. **Blog companion** — text version with video embedded
8. **Email newsletter** — "watch this" + link

## Customer case study

**Core:** Written case study or recorded interview.

**Derivatives:**
1. **Instagram carousel** — before/after visuals with metrics
2. **Instagram Reel** — customer speaking to camera (30-60s)
3. **LinkedIn post** — "{Customer} saved {N} using {product}" story format
4. **Twitter quote card** — best quote from customer
5. **Blog post** — full written version
6. **Email nurture** — "How {customer} did it" deep dive
7. **Landing page testimonial** — excerpted quote + photo

## Data report / survey results

**Core:** Research or proprietary data.

**Derivatives:**
1. **Instagram carousel** — one chart per slide
2. **Twitter thread** — key stats unraveled
3. **LinkedIn article** — longer analysis
4. **Report PDF** — gated download for lead gen
5. **Webinar** — present findings live
6. **Email series** — one insight per email over 3-5 emails
7. **Press release** — if newsworthy enough

## Podcast episode

**Core:** 30-60 min audio.

**Derivatives:**
1. **Audiogram for Twitter/LinkedIn** — 30-60s clip with waveform animation
2. **TikTok/Shorts** — best 30s moment with captions
3. **Instagram carousel** — quote + takeaway slides
4. **Blog post** — transcript-based article
5. **Show notes email** — key timestamps + links
6. **YouTube upload** — video version if recorded with cameras

## Webinar / workshop

**Core:** 45-60 min live session.

**Derivatives:**
1. **YouTube upload** — full recording
2. **3-5 Shorts** — best moments
3. **Blog post** — written recap with screenshots
4. **Instagram carousel** — top 5 slides from deck
5. **Twitter thread** — live-tweet summary
6. **LinkedIn article** — speaker's perspective
7. **Email follow-up** — to attendees with slides + recording

## Single image / photo

**Core:** One great photo (product, BTS, customer, event).

**Derivatives (lower leverage — don't over-stretch):**
1. **Instagram post** — primary destination
2. **Twitter** — same image, adjusted caption
3. **LinkedIn** — if professional/relevant
4. **Facebook** — same image, FB-style caption
5. **Story / Reels cover**

## The "3x framework"

For every piece of content, ask:
1. **Can I cut it shorter?** (long video → shorts, long post → carousel)
2. **Can I change the format?** (video → blog, audio → quote image)
3. **Can I share the process?** (BTS of making the core piece = new entertaining post)

Each yes = a new derivative.

## Batching schedule

- **Week 1:** Produce 2 core pieces (blog + video, e.g.)
- **Week 2:** Derive 10-15 social posts from those cores
- **Week 3:** Schedule all derivatives across 3-4 weeks
- **Week 4:** Produce next 2 cores

This way, 50% of time goes to cores, 30% to derivatives, 20% to community management.
