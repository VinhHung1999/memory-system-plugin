# Strategy Ideation Prompt Template

From Course 7 M4 m4_04 "Kickstart your marketing strategy ideas with AI".

Use when facing blank-page block on a new campaign or launch plan.

## Full T-C-R-E-I prompt

```
**Task (T):** Draft an initial marketing plan for launching {product_or_feature} to {audience} over {timeframe}. Include strategy, channel mix, content themes, KPIs, and 3-week execution outline.

**Context (C):**
- Company: {company_name} — {brief_business_description}
- Launching: {product_or_feature} — {1_sentence_value_prop}
- Target audience: {audience_segment} (pulled from prior audience research if available)
- Funnel stage focus: {awareness / consideration / conversion / loyalty}
- Budget: {amount or "scrappy/moderate/generous"}
- Team size: {1_person / small_team / enterprise}
- Timeline: {launch_date and campaign_duration}
- What's worked before: {past_channel_win_if_any}
- What hasn't: {past_channel_loss_if_any}

**References (R):**
- Brand voice — 3-5 adjectives: {voice_adjectives}
- Brand mission in one sentence: {mission}
- A competitor doing something we'd like to emulate: {competitor_name} — {what_they_do}
- A competitor we want to differentiate from: {other_competitor} — {what_we_avoid}

**Evaluate (E):**
Before finalizing the plan:
1. Are all proposed channels realistic for our team size and budget? Flag any that aren't.
2. Which single channel has the best effort/impact ratio? Justify.
3. What's the #1 risk to this plan? (audience mismatch? cadence? budget?)
4. Is the KPI selection overly vanity-metric-heavy?

**Iterate (I):**
After initial draft:
- "Take the #1-impact channel and detail a 4-week content calendar"
- "Rewrite the KPIs as specific SMART goals"
- "Cut this plan to only what I can realistically execute as a solo marketer"
```

## How to use

1. Fill in. Don't skip Context — more context = more tailored output.
2. Run it. Expect a solid first draft.
3. Iterate. Always do at least 1 iteration — first draft is 70%.

## Expected output structure

- **Strategy summary** (positioning, pitch)
- **Channel mix table** with effort/impact
- **Content themes** (4-6 buckets)
- **KPIs** (primary + secondary)
- **3-week execution outline** (week 1/2/3 actions)
- **Risk flags**

## Common failure modes

- **Plan too ambitious** → Context skipped team size + budget. Add constraints.
- **All-channel list** → No prioritization forcing function. Ask "pick ONE to focus 80% effort."
- **KPIs are impressions/reach** → Vanity metrics. Explicitly ask for business-outcome metrics (revenue, signups, retention).
