# Email Marketing Benchmarks

From Course 4 Module 4. Industry averages (2023-2024) + vertical adjustments.

## Universal benchmarks

| Metric | Benchmark | What to do if below |
|---|---|---|
| **Open rate** | 20-25% | Rework subject lines; clean list (hard bounces) |
| **Click-to-open rate (CTOR)** | 10-15% | Improve body content, single CTA focus |
| **Click-through rate (CTR)** | 2-5% | Stronger primary CTA, reduce distractions |
| **Conversion rate** | 1-5% | Landing page mismatch OR friction at conversion step |
| **Unsubscribe rate** | <0.5% | Cadence too high OR list quality poor |
| **Complaint rate (spam)** | <0.1% | Permission issue — check opt-in flow, sender auth |
| **Bounce rate (hard)** | <0.5% | Clean list; remove hard bounces immediately |
| **Bounce rate (soft)** | <2% | Usually temporary; retry, remove if 3+ soft bounces |
| **List growth rate** | +2-5% per month | Add signup forms, lead magnets; reduce unsubs |
| **ROI** | 42:1 ($42 per $1 spent) | Reassess targeting + offer, not just creative |

## By vertical

### E-commerce
- Open rate: 18-22% (slightly below avg due to promotional heaviness)
- CTR: 2.0-3.5%
- Conv rate: 1-3% (lower ticket = higher conv, higher ticket = lower)
- ROI: 45:1 typical, 70:1+ for well-segmented brands

### SaaS / B2B
- Open rate: 22-28% (smaller list, higher relevance)
- CTR: 2.5-4%
- Conv rate: varies widely by definition (trial start, demo book, upgrade)
- ROI: harder to measure; focus on pipeline generated vs spend

### Nonprofit
- Open rate: 25-30% (high mission alignment)
- CTR: 3-5%
- Conv rate (donation): 0.5-2%
- ROI: measured per dollar raised per dollar spent

### Media / Content
- Open rate: 25-30%
- CTR: 4-8% (click-centric by nature)
- Conv rate (newsletter signup, subscription): 1-3%

### Retention / transactional
- Open rate: 40-60% (order confirmations, password resets)
- CTR: varies, often high (15-30%)
- These aren't marketing — exclude from marketing benchmarks

## By email type

| Type | Typical open | Typical CTR |
|---|---|---|
| Welcome | 45-60% | 10-15% |
| Promotional | 15-25% | 2-4% |
| Newsletter | 22-28% | 3-6% |
| Retention | 20-35% | 5-10% |
| Acquisition | 12-20% | 1-3% |

## Red flags

- **Sudden open rate drop** — likely deliverability issue (check sender reputation, SPF/DKIM/DMARC, Gmail/Outlook blocking)
- **Spam complaint rate > 0.1%** — immediate action: pause send, investigate, consider list purge
- **Bounce rate > 5%** — list hygiene problem; your ESP may throttle you
- **Unsubscribe rate > 2%** — offering is mismatched OR cadence is too high
- **CTR but no conv** — landing page issue, not email issue

## How to use these

1. Set target = benchmark (universal or vertical-adjusted)
2. Every send: compare actual vs target
3. Investigate ANY metric 20%+ below benchmark
4. Every quarter: recompute rolling 90-day averages to track trends

The benchmarks are floors — excellence is 25-50% above these. Top-decile brands hit 40-50% open rates and 8-12% CTR via ruthless relevance + list quality.
